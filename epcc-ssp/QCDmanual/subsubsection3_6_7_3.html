6.7.3 Creating your own maps

Maps are best created by using a map that has already been created and adding and deleting modules as necessary. It is recommended that when creating maps small sets of test data are used. Explorer provides a module called GenLat which creates data for you. The best way to proceed is to add in modules one at a time and check what is created at each stage. The help option on the modules will aid the user in connecting the input and output ports correctly. The user manual [4] has information about maps and some examples on creating simple maps, however through experience the user should be able to create maps of their own with relative ease.
