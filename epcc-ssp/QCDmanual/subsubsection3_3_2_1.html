3.2.1 BOB

BOB stands for Brick Of Bytes which is the data format (see section 5.2) for this visualisation package. BOB was created for use in the American Army high performance Computing research center. The key feature was to allow images of three dimensional volumes to be viewed quickly and interactively. Although there is little that can be done apart from viewing all of, or sections of, a 3D volume and rotation, BOB is a very useful tool. It has stability and speed firmly on its side. It enables the user to browse through data files very quickly, and therefore choose those which require more detailed viewing. This is where Explorer comes in (see section 3.2.2).
