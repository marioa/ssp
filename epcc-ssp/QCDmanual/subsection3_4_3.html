4.3 Option 3 - Translating into probabilities

To aid the user in locating the data in the visualisation stage and to give physical meaning to the data values, it is necessary to convert the data into probabilities. In otherwords scaling the data so that the total sum of all the data values is one, one being 100%probable.

Once the user has selected this programme s/he will be asked for the names of the files to be converted into probabilities. Filenames should be in the form specified in section 4.4.
