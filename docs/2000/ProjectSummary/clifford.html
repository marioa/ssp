Jini-Grid

Student

Ben Clifford, Queen Mary and Westfield College, London

Supervisors

Martin Westhead, EPCC

Scott Telford, EPCC

The aim of this project is to use Java technology and Jini in particular to build a small experimental prototype metacomputing system within EPCC. JavaGrid would provide basic 'Grid' services such as the opportunity to remotely execute jobs on the Sun cluster, BOBCAT and possibly Lomond. Additional services which could be investigated and possibly implemented include visualisation, job monitoring, security/authentication, job accounting, job scheduling etc.
