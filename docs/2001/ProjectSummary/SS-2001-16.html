Project Title

 Testing and Development of the DPD code

Project Proposer(s)

 Lorna Smith + Alexander Wagner (Physics)

Project Details

This project will focus on the parallel DPD code developed by the physics department. The code is currently written in MPI, however further work is required to document and test the code on the T3E. In addition some code development may be anticipated. Finally, if time permits, the project will focus on the parallelism of a serial version of the code using OpenMP.

Work Plan

Week 1 Courses

Week 2,3 Familiarisation with the code

Week 4,5,6 Documenting the code + running specific tests

Week 7,8,9 Further development, e.g. code development, optimisation and if time permits parallel OpenMP code development.

Week 10 Writting up

Computing Skills Required

Good fortran programming skills are essential. A knowledge of MPI and/or OpenMP would be adventageous. Finally this project would be best suited to a student with a background in physics, who may have more interest in the science of the code.

Training Required

Intro to HPC, MPI and OpenMP
