Project Title

 Feedback guided loop scheduling

Project Proposer(s)

 Mark Bull

Date of proposal

 2001/02/09 11:32:46

Project Details

The aim of this project is to do further investigations and experiments in feedback guided loop scheduling. These are algorithms which attempt to find an efficient irregular block schedule over successive executions of a parallel loop. The student will take the existing algorithms and (using existing code implementations) explore some variants and improvements. Benchmarking will be done both in simulation, and on Sun HPC systems. There may be an opportunity to test the algorithms in a real application.

Work Plan

Week 1: Training
Week 2: Familiarisation with papers and codes.
Week 3-4: Implementing enhanced algorithms.
Week 5: Implement new test cases.
Week 6-9: Benchmarking and testing.
Week 10: Writing report
