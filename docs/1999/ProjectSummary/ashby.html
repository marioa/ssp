Parallel Rasterisation For GIS

Student

Thomas Ashby, University of Edinburgh

Supervisors

Connor Mulholland, EPCC

Terry Sloan, EPCC

The proposed project will be viewed as a valuable extension to the current collaborative work being carried out between EPCC and GIS-PAL. This work addresses the application of parallel processing to vector topological data. The SSP work submitted involves the coding and testing of the Rasterisation Component of the Vector to Raster conversion. The purpose of the component is to produce a data representation known as Raster (in which space is represented by a regular tesselation) from the Vector input data.

The final report for this project is available here.
