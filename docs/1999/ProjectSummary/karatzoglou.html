Java Grande is a community initiative led by NPAC and Sun to promote the use of Java for so-called "Grande" applications. A Grande application is an application which has large requirements of memory, bandwidth and processing power. Grande applications include computational science and engineering codes, as well as large scale database applications and business and financial models. EPCC are leading the benchmarking initiative to develop a suite of benchmarks to measure different execution environments of Java against each other and native code implementations.

This project will involve the development of the current benchmark suite, with particular emphasis on developing parallel benchmarks.
