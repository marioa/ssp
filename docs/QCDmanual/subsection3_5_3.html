5.3 Viewing your data

When BOB starts up it views the image in the data file at the top of the list of filenames. To view another file simply click over the filename with the left mouse button and BOB will display this file. There is also a scroll bar on the right of the list of files that can be used in the normal way.

When the mouse is in the section of the main window that contains the image the mouse buttons have various functions

  • Left mouse button - holding this down and moving will rotate the image
  • Middle mouse button - holding this button down and moving will move the image in the plane of the screen
  • Right mouse button - holding this button down will cause the image to zoom towards or away from the user depending on whether it is moved up or down.

The image can also be continuously rotated by letting go of the left mouse button whilst still moving the mouse. The image will keep moving in that direction until the user clicks over it.
