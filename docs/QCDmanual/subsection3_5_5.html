5.5 Where to go for more information

There is a user manual provided for BOB called ``All About Bob - A tool for browsing 3D data sets'' [2]. This can be found in the following directory:

nevis/Tools/Evaluation/GVL/vis_tools/doc/allbob.show.PS.Z

and must be copied into the users directory and uncompressed to view as follows:

uncompress allbob.show.PS.Z

and can be viewed using ghostview:

ghostview allbob.show.PS &
