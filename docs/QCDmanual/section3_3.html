3 Facilities available

This section contains an explanation of the facilities that have been provided. There will be a description of the programmes that have been created to aid the visualisation, as well as an introduction to the visualisation packages.

The various facilities included in this manual are intended to take data from the UKQCD group in its initial form (IEEE floating point, 32-bit, single precision, binary representation. X should be the fastest varying axis, followed by Y, then Z). If the data is not in this form then it will be necessary to convert it for use of the programmes that have been provided. If only the visualisation packages are to be used there are various data import modules available which will be mentioned later, however the facilities covered in this manual assume the data type which results from use of the programmes mentioned in the following section.
