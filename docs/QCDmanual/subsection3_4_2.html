4.2 Option 2 - Averaging data

This programme will average data over a number of files. The purpose of this is to create data sets that have greater physical meaning.

Once the user has selected this option from the menu they will be required to enter in the filenames in the form explained in section 4.4. The programme will then average the data contained in these files, writing the result to the output file specified by the user.
