6.2 Explorer Log

The Explorer Log will tell you when there are errors. The best thing to do if explorer has some errors you don't understand, especially if modules start being deleted, is to Quit Explorer and restart.

In general if you understand the errors then do something about them. If you don't, don't worry...Explorer tends to crash frequently, especially when experimenting with viewing large data sets, consequently it is advisable to save maps frequently, preferably in the users directory.
